Other Tools

In the Filter tab, there are a couple other tools that are added by the Blockbench-MDL Exporter

Set all MDL3 Group

This tool will set every group to an MDL3 Group

Remove all MDL3 Groups

This tool will unset every group that was an MDL3 Group
