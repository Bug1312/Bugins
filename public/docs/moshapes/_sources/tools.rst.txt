Tools
=====

Tools can be found by searching for them or using the `Mo' Shapes Bar Menu <./barmenu.html>`__

Load Shapes
-----------

This tool is used for loading or reloading shapes

Remove Shapes
-------------

This tool is used for removing all custom shapes

Mo' Shapes Live
---------------

This tool will set a property in your format that will update run the load shapes tool every two seconds. It does have a warning though, if there are multiple shapes in the current project then it can reduce frame rate drawing all of the shapes repeatedly.
