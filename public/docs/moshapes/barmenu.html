Bar Menu¶

This is an easy access menu for creating shapes as well as using Mo’ Shapes tools.

Tools included in the latest version are as follows:

  • Load Shapes

  • Remove Shapes

  • Mo’ Shapes Live

  • Add Cone Indicator

  • Add Cylinder Indicator

  • Add Icosahedron Indicator

  • Add Knot Indicator

  • Add Octahedron Indicator

  • Add Ring Indicator

  • Add Sphere Indicator

  • Add Tetrahedron Indicator
